A collection of standalone C++ players for sequential animation, emoji slideshow, and image gallery – with real-time zoom, pan, and play/pause controls.

## Included Programs

- **SequencePlayer** – displays a counting sequence with zoom/pan.
- **EmojiPlayer** – cycles through a library of emojis.
- **ImagePlayer** – loads all images from a selected folder and plays them as a slideshow.

## How to Compile (Windows)

Use MinGW (g++):

```bash
g++ -o ImagePlayer.exe ImagePlayer.cpp -lgdi32 -lgdiplus -lshell32 -lole32 -mwindows
